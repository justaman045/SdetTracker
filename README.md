## Features

- **Daily Logging** — Record hours studied, topics covered, confidence level, and blockers
- **Study Roadmap** — Pre-built SDET curriculum across 6 phases (Core Java → Selenium → API Testing → Appium → Frameworks → Advanced)
- **Interview Prep** — Curated interview questions organized by topic with difficulty ratings
- **Notes** — Markdown-based study notes with real-time Firebase sync
- **Progress Dashboard** — Completion percentage, streaks, total hours, and motivational milestones
- **Auth** — Firebase Authentication with Google sign-in

## Curriculum Phases

| Phase | Topic | Duration |
|-------|-------|----------|
| 1 | Core Java Deep Revision | Month 1 |
| 2 | Selenium WebDriver Mastery | Month 2 |
| 3 | API Testing (REST Assured + Postman) | Month 3 |
| 4 | Mobile Automation (Appium) | Month 4 |
| 5 | Test Framework Design Patterns | Month 5 |
| 6 | Advanced Concepts & CI/CD | Month 6 |

## Getting Started

```bash
# Clone
git clone https://github.com/justaman045/learning-tracker.git
cd learning-tracker

# Install
npm install

# Set up environment
cp .env.example .env
# Fill in Firebase config

# Run dev server
npm run dev
```
